# Changelog

### Added
- Initial Prisma schema with multi-tenancy
- Core models: Tenant, User, Session, ApiKey
- Role and permission models
- Approval workflow models
- Sandbox environment models
- SIEM configuration models
- SSO configuration models (SAML, SCIM)
- Organization model (first feature)
- Audit log model
- Tenant-isolated client wrapper
- Migration infrastructure
